Esta opção permite fazer o backup da base de dados do app para posterior recuperação em um outro aparelho ou no mesmo no caso de reinstalação do app. O arquivo será armazenado no diretório "Android/data/com.spendingmanagerapp/files", que deve ser copiado para um outro diretório logo após o backup, pois se o app for desinstalado o diretório do mesmo será apagado juntamente com a base.
