O upload da base de dados permite utilizar uma base previamente carregada. Para realizar o mesmo, primeiramente é preciso mover o arquivo "base.bd", que foi salvo na opção de Download, para a o diretório "Android/data/com.spendingmanagerapp/files", após isso basta fazer o upload, fechar o app e entrar novamente que os dados pré cadastrados já estarão lá novamente.
